 # Desparcha234

Desparcha234 es un videojuego multijugador local para dispositivos móviles y tablets, inspirado en los juegos tradicionales colombianos. 

![Grabación-de-pantalla-2025-11-11-233008](https://github.com/user-attachments/assets/d83f6ef2-b993-4beb-89e0-462dfdca2bcf)


El juego incluye varios minijuegos representativos, entre ellos:

* Canicas

* Catapis

* Tejo

* Carrera de sacos

* La lleva

* Tingo Tingo Tango

* La lleva congelada (minijuego secreto desbloqueable)
